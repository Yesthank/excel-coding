# ⚡ Excel Sequential Data Copier (엑셀 순차 복사기)

대량의 엑셀 데이터를 **단축키(Shift) 하나로 순서대로 복사**해주는 자동화 도구입니다.
사내 보안 환경(VBA 객체 제한 등)에서도 문제없이 작동하도록 설계되었습니다.

## 📌 주요 기능
* **초간편 입력:** `왼쪽 Shift`를 누르면 데이터가 클립보드에 복사됩니다. (`Ctrl+V`로 붙여넣기만 하면 끝!)
* **실수 방지:** `오른쪽 Shift`를 누르면 이전 데이터로 되돌아갑니다.
* **자동 반올림:** 지저분한 소수점을 원하는 자릿수로 깔끔하게 잘라서 복사합니다.

## 🚀 사용 방법
1. 엑셀에서 `Alt + F11`을 누릅니다.
2. 상단 `삽입`-`모듈`탭을 클릭하고 `code.txt` [code.txt](code.txt)에 있는 코드를 복사, 붙여넣기 합니다.
3. 엑셀에서 `Alt + F8`을 누릅니다.
4. `StartSequentialCopy`를 선택하고 **[실행]**을 누릅니다.
5. 준비 완료 메시지가 뜨면 입력을 시작합니다.
   - **`[왼쪽 Shift]`**: 복사하기
   - **`[오른쪽 Shift]`**: 뒤로 가기
   - **`[ESC]`**: 종료하기

## ⚙️ 설정 수정 방법 (초보자용)
매크로 편집창(`Alt + F11`)을 열면 코드 상단에 **[사용자 설정 구역]**이 있습니다.
이 부분의 숫자나 문자만 바꾸면 입맛대로 수정할 수 있습니다.

```vba
' ==========================================================
' ⚡ [사용자 설정 구역] 이 부분만 수정하세요! ⚡
' ==========================================================

' 1. 데이터 위치 설정
Const START_ROW_NUM As Long = 1      ' 데이터가 몇 번째 줄부터 시작하나요?
Const START_COL_CHAR As String = "B" ' 시작 열 (예: A, B, C...)
Const END_COL_CHAR As String = "G"   ' 끝 열



' 2. 반올림 설정 (소수점 몇 째 자리까지 남길지)
Const DIGITS_DEFAULT As Integer = 3  ' 기본적으로 소수점 3자리까지 남김 (0.###)
Const DIGITS_SPECIAL As Integer = 5  ' 특별한 열은 소수점 5자리까지 남김
Const SPECIAL_COL_CHAR As String = "E" ' 특별 규칙을 적용할 열 (예: E열)
